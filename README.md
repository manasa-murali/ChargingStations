# ChargingStations

## Application to find electric vehicle charging stations for the given location data

* ### Architecture followed - MVVM
* ### Language - Kotlin
* ### Libraries
    * Retrofit - Network Request
    * HILT - Dependency Injection
    * RxJava, Flow - Reactive Approach
    * Gson - Serialization
    * Mockk - Unit test
